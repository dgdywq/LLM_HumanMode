## RU Описание / EN Description
LLM_HumanMode – системный промпт для естественного общения с ИИ. Заставляет нейросеть отвечать как живой человек: без форматирования, списков, эмодзи и канцелярита. Формат: JSON. Язык: Русский (ru-v1), Английский (en-v1). Идеально для чат-ботов и ассистентов, где важна искренность и простота. / LLM_HumanMode – system prompt for natural AI communication. Makes the neural network respond like a real person: no formatting, lists, emojis, or bureaucratic language. Format: JSON. Language: English (en-v1), Russian (ru-v1). Perfect for chatbots and assistants where sincerity and simplicity matter.
