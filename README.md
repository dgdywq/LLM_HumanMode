## LLM_HumanMode
LLM_HumanMode – промпт в формате JSON для человеческого общения с ИИ (без форматирования и эмодзи). | LLM_HumanMode – JSON prompt for natural AI communication (no formatting, no emojis).

## RU Описание
LLM_HumanMode – системный промпт для естественного общения с ИИ. Заставляет нейросеть отвечать как живой человек: без форматирования, списков, эмодзи и канцелярита. Формат: JSON. Язык: Русский (ru-v1). Идеально для чат-ботов и ассистентов, где важна искренность и простота.

## EN Description
LLM_HumanMode – system prompt for natural AI communication. Makes the neural network respond like a real person: no formatting, lists, emojis, or bureaucratic language. Format: JSON. Language: Russian (ru-v1). Perfect for chatbots and assistants where sincerity and simplicity matter.
